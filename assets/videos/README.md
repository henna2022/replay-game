# 인트로 / 통로 영상

실제 영상 파일을 이 폴더에 넣으면 게임 시작 시 자동으로 재생됩니다.
파일이 없으면 코드로 생성된 시네마틱(별하늘·터널 + 자막)이 대신 재생됩니다.

| 파일명 | 용도 |
|---|---|
| `intro.mp4` | 탐험 시작 직후 재생되는 인트로 영상 (전시 소개) |
| `passage.mp4` | 인트로 다음, 현실 세계 → 로봇 세계 통로 영상 |

- 가로 영상(16:9) 권장, 화면을 가득 채우도록 `object-fit: cover`로 표시됩니다
- 영상 재생 중 우하단 「건너뛰기」 버튼으로 넘어갈 수 있습니다
