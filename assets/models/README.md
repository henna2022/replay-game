# 3D 모델 슬롯

이 폴더의 GLB 파일들이 게임 속 전시물로 자동 배치됩니다 (meshopt 압축, 웹 최적화 완료).

| 파일명 | 대상 | 상태 |
|---|---|---|
| `g1.glb` | 미션1 로봇 댄스 존 — G1 휴머노이드 | ✅ 배치됨 |
| `robotdog.glb` | 미션1 로봇 댄스 존 — 로봇개 | ✅ 배치됨 |
| `liku.glb` | 미션3 마음나누기 존 — 리쿠 | ✅ 배치됨 |
| `lovbot.glb` | 미션3 마음나누기 존 — 러봇 | ✅ 배치됨 |
| `balu.glb` | 미션3 마음나누기 존 — 발루 | ⏳ 파일 넣으면 자동 교체 |
| `ameca.glb` | 아메카 전시 (맨 마지막) | ✅ 배치됨 |

## 동작 방식

- 파일이 있으면 자동으로 크기와 바닥 위치를 맞춰 배치하고, 코드로 만든 절차 생성
  모델을 숨깁니다. 파일이 없으면 절차 생성 모델이 표시됩니다.
- 애니메이션 클립이 없는 정적 모델은 코드가 움직임(댄스 바운스, 두리번거림 등)을 입힙니다.

## 새 모델 추가/교체 시 웹 최적화

원본 GLB가 10MB를 넘으면 아래 명령으로 압축하세요 (10배가량 줄어듭니다):

```bash
npx @gltf-transform/cli optimize input.glb output.glb \
  --compress meshopt --texture-compress webp --texture-size 1024
```
