# 미스터리 놀이터 「RE:PLAY」 — 3D 아카이브 게임

서울로봇인공지능과학관 엔터테크 기획전시 「미스터리 놀이터」를 3D로 아카이빙한 웹 게임입니다.
기획 대본(해설 투어)의 순서 그대로, 현실과 로봇 세계를 잇는 통로를 지나
10가지 미션을 차례로 완수하고 황금 열쇠를 찾아 현실로 귀환합니다.

## 실행 방법

로컬 서버로 실행해야 합니다 (ES 모듈 사용):

```bash
cd replay-game
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

갤럭시 탭에서는 같은 Wi-Fi에서 `http://<컴퓨터IP>:8000` 접속,
또는 GitHub Pages / Netlify에 폴더째 배포하면 됩니다.

## 게임 흐름

1. **인트로 영상** → **통로 영상** (파일이 없으면 자막 시네마틱으로 대체, 건너뛰기 가능)
2. **ENTRANCE에서 시작** — 플로어맵의 입구 통로 위치
3. **10가지 미션 순차 진행** (초록 빛기둥이 다음 미션 안내):
   로봇 댄스 → 로봇팔스튜디오 → 소형로봇전시 → AI사운드 뮤드럼 → 디지털 아바타
   → 버추얼 아이돌 → 축구 슛팅 → 운과 확률(황금 열쇠) → 아메카 → 데이터 분석실
4. **통로로 귀환** — 황금 열쇠로 엔딩
- 축구 슛 데이터는 데이터 분석실의 포지션 추천에, 아바타는 버추얼 아이돌 팬미팅에 이어집니다
- **90초 무입력 시** 경고(75초) 후 처음 화면으로 자동 복귀 (키오스크 운영용, 진행은 저장됨)

## 조작

- **태블릿**: 왼쪽 아래 드래그 = 이동(가상 조이스틱), 오른쪽 드래그 = 시점 회전, 전시물 탭 = 미션
- **PC**: WASD/방향키 = 이동, 마우스 드래그 = 시점 회전, 클릭 = 미션
- 전시물에 가까이 가면 하단에 체험 버튼이 뜹니다

## 콘텐츠 커스터마이징

- `assets/photos/zone-01.jpg`~`zone-11.jpg` — 실제 전시 사진을 넣으면 존 액자에 자동 표시
- `assets/videos/intro.mp4`, `passage.mp4` — 실제 인트로/통로 영상을 넣으면 자동 재생
- `js/config.js` — 존 위치·벽 구조(WALLS)·미션 순서(SEQUENCE)·타임아웃(90초) 설정

## 구조

- `js/config.js` — 존 데이터, 벽 세그먼트(플로어맵 재현), 진행 순서, 타임아웃 설정
- `js/world.js` — 전시홀 + 벽 + 11개 전시물 저폴리 모델 + 안내 비컨
- `js/missions.js` — 대본 기반 10개 미션 (미니게임 + 퀴즈 + 생각 문답)
- `js/cinema.js` — 인트로/통로 영상 스테이지 (영상 없으면 자막 시네마틱)
- `js/player.js` — 1인칭 터치/키보드 컨트롤 + 벽 충돌

진행 상황은 브라우저 localStorage에 자동 저장됩니다. (기록장 → 기록 초기화)
